# Title text adder Web APP


## Useful deploy commands

```shell
mvn tomcat7:deploy
```
```shell
mvn tomcat7:undeploy
```
```shell
mvn tomcat7:redeploy
```

## Make curl request

```shell
curl -X GET http://localhost:8080/tta/titles/test
```

```shell
curl -X GET http://localhost:8080/addsTitle.htm
```
